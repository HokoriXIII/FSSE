FalloutShelter save editor

Easy to use save editor for Fallout Shelter.

Information

This app is an Android only editor for Fallout Shelter.
It works in BlueStacks and it does not require root.
But the most important thing: it does NOT require any technical skills:
  • No computer required for the editing process, everything works on your device.
  • Vault encrypting and decrypting not required.
  • No JSON editing required.
Current version: 1.5.3
Currently supported game version: 1.8

How to install?

  • 1) Download the editor APK using the download link or the QR code above.
  • 2) Enable "Unknown sources" on your device. Settings->Security.
  • 3) Copy the APK to your device.
  • 4) Browse the APK using a file manager and install it.

How to use?

  • 1) Start the editor app on your device.
  • 2) Select the vault you'd like to edit.
  • 3) Modify things.
  • 4) Save the vault.

What can you edit?

  • Caps, power, food and water.
  • Stimpacks and radaways per dweller and vault.
  • Vault storage (add items/pets).
  • Add legendary and rare dwellers to your vault.
  • Create custom dwellers.
  • Change vault number.
  • Copy/multiply dwellers.
  • Add items (weapons and outfits) to your vault's storage.
  • Complete objectives.
  • Edit pets (their names and bonus effects).
  • Edit "Survival Guide" pages.
  • Fix time glitched vaults.
  • Edit dwellers:
    • Name and level
    • S.P.E.C.I.A.L.
    • Hair, sking and outfit color
    • Hair style and facemask
    • Inventory
  • + Many many more.

Remote editing (use your Android device to edit PC saves): (Click to open)

  • 1) Start the windows editor.
  • 2) Open a vault and press "Remote edit".
  • 3) Start FSSE (min v1.4.1) on your Android device.
  • 4) Select remote edit and enter the IP shown in the editor or scan the QR code.
  • 5) Edit the vault on your phone.
  • 6) Save the changes and wait until the Windows editor and FSSE do their job.
Note: Your Android device and PC must be in the same local network.
Note #2: If the editors are unable to save or load, try again.

Modded game (Click to open)

Mods: Note: These mods can be enabled and disabled using both editors or by manually editing the config file.
By default every mod is enabled in the config!
  • Unlimited dwellers: There's a reason behind the 200 dwellers limit but if you want to experiment with an ENORMOUS vault, well... you can! The modded version has the maximum possible limit of dwellers which is 231 (2,147,483,648)! (no_vault_dweller_limit)
  • No pet verification: The original version of FalloutShelter checks if a pet has its correct bonus type and value which makes it un-editable. Using the modded version it's possible to remove this restriction. (allow_pet_bonus_modification)
  • Room destruction: This mod makes it possible to destory any room (so unexpected things may happen). (destroy_any_room)
  • Exploring limits: Maximum dwellers in wasteland removed (original: 25), also removed the radaways and stimpacks cap. (remove_wasteland_\mrhandy_cap\mrhandy_caps_limit\dweller_limit)
  • Pet limits: Maximum pets per vault and per room removed. (remove_pet_count_caps)
  • MrHandy limits: One MrHandy per floor limit removed. (remove_1_mr_handy_cap)
  • Rush mods: Rushes are instant and they never fail. (rush_instant, rush_never_fails)
  • No living quarter limits: Removes the dweller limits based on living quarter counts. (no_vault_dweller_limit)
  • Instant wasteland quests: Get instant random wasteland quests. (instant_wasteland_quests)
  • Unlimited resources: Makes power, water and food unlimited. (unlimited_resources)
  • Skip more than one objective: Skip as many objectives as you want. (skip_more_objectives_per_day)
  • Remove nuka caps limit: Removes the 1mil nuka caps limit. (remove_storage_cap)
  • Remove storage size limit: No need to build any storage rooms, it's infinite. (remove_storage_cap)

Use it at your own risk! Note: Some mods may act differently on Android than Windows.

Android download and instructions (Click to open)

Direct download / Virus scan

Version: 1.9

How to install?

  • 1) Download the modified game APK from the link above.
  • 2) !! Backup your saves / vaults, you can use the editor to make backups or goto "/sdcard/Android/data/com.bethsoft.falloutshelter/files" and save your "Vault#.sav" files.
  • 3) Backup the OBB of the game (located here: "\sdcard\Android\obb\com.bethsoftfalloutshelter\"). You can do this by renaming the "com.bethsoftfalloutshelter" folder to something else.
  • 4) Delete the current version of the game from your device (this will delete all your save files and the OBB).
  • 5) Install the modified version.
  • 6) Rename the OBB folder to it's original name (again here: "\sdcard\Android\obb\com.bethsoftfalloutshelter\"). This is required because newer versions of the game does not download the OBB and if the game can't find it, it will just crash.
  • 7) Restore the save files.
  • 8) Copy the contents of the "sdcard" folder to the sdcard of your device. Make sure the "Android/data/com.bethsoft.falloutshelter/files" folder exists.
  • 9) Using the guide below configure the mods and enjoy.

Enable/Disable mods (also first setup)

  • 1) Download FSSE for Android.
  • 2) Select "Setup mods".
  • 2.1) Press "Yes" if the app asks if you want to download default config files.
  • 3) Enable and disable mods as you like.
  • 4) Restart the game and enjoy.
Or manually:
  • 1) Goto "sdcard/Android/data.com.bethsoft.falloutshelter/files/Mods".
  • 4) Edit the "mods_config.json" file as you like.
  • 5) Restart the game and enjoy.

PC download and instructions (Click to open)

Direct download / Virus scan

Version: 1.9

How to install?

  • 1) Download the file from above and unpack it.
  • 2) Locate the installation folder of the Bethesda Launcher (right click on the icon on your desktop and you'll be able to see the folder).
    You can use the editor to locate the folder ("Open folder"->"Installation folder" menu).
  • 3) In the folder of the launcher goto: "games/FalloutShelter/FalloutShelter_Data/".
  • 4) Extract the downloaded zip and place the two folders into the "FalloutShelter_Data" folder.
  • 5) Start the game.
  • 6) Using the guide below configure the mods and enjoy.

Enable/Disable mods (also first setup)

  • 1) Download FSSE for Windows.
  • 2) Select "Mods" (near the "File" menu).
  • 2.1) Press "Yes" if the app asks if you want to download default config files.
  • 3) Enable and disable mods as you like.
  • 4) Restart the game and enjoy.
Or manually:
  • 1) Goto the installation folder of Fallout Shelter and goto "FalloutShelter_Data" then "Mods".
  • 4) Edit the "mods_config.json" file as you like.
  • 5) Restart the game and enjoy.
Note: If the mods aren't working try starting the game using it's exe (FalloutShelter.exe), because the launcher might know that the files changed and it would download the original ones.
